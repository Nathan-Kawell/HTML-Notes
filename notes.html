Fully qualfied or abssolute URL starts with http, https, or file. Depending on whether you're accessing something locally, on the web, and if it's secure or not.

A link must be made with an A element, or Anchor. Because this anchors the link to the page. An A element must include an Attribute to function properly.

An Attribute should start with a name (i.e. href) followed by a value. For example, YouTube. Href is hypertext refrence.

Image element, or img, has to have 2 Attributes. First is src (source), which is the url of the image, and alt (alternate), which is a text description of the image. Img is an empty element because it has no textual data and has no close tag.
